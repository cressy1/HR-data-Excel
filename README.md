# HR Analytics Dashboard (Excel)

## Project Overview

This project is an interactive **HR Analytics Dashboard** developed in Microsoft Excel to analyze employee attrition and workforce demographics. Using a dataset containing **1,470 employee records**, the dashboard provides valuable insights into employee turnover, workforce composition, and key factors influencing attrition.

The goal of this project is to help HR professionals and business leaders understand workforce trends, identify high-risk employee groups, and support data-driven decision-making for employee retention and workforce planning.



## Project Objectives

* Analyze overall employee attrition across the organization.
* Monitor key HR metrics through an interactive dashboard.
* Identify departments and job roles with the highest attrition.
* Examine how age, gender, education, marital status, and overtime relate to employee turnover.
* Evaluate workforce demographics to support HR planning.
* Provide actionable insights that can improve employee retention strategies.



## Dataset

The dataset contains information for **1,470 employees** and includes attributes such as:

* Employee ID

* Job Role

* Environment Satisfaction

* Years in Current Role

* Total Working Years
* Training Times Last Year
* Attrition Status



## Dashboard Features

The dashboard includes interactive reports and visualizations for:

* Overall HR KPIs

  * Total Employees
  * Current Employees
  * Employees Who Left
  * Attrition Rate

* Attrition Analysis by:

  * Department
  * Job Role
  * Gender
  * Age Group
  * Education
  * Marital Status

* Employee Rating Analysis

* Workforce Demographics

* Interactive filtering for deeper analysis



## Key Business Questions Answered

* What is the overall employee attrition rate?
* Which departments experience the highest turnover?
* Which job roles are most affected by attrition?
* Does attrition differ between male and female employees?
* Which age groups are most likely to leave the organization?
* How does education level influence attrition?
* What impact does marital status have on employee turnover?
* Does overtime contribute to higher attrition?
* Which employee segments require the most attention from HR?
